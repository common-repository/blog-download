=== Blog Download ===

Contributor: Rodrigo Villanueva Ceballos - zencherry.com

== Description ==


Plugin for downloading WordPress posts in PDF, DOC and HTML formats.
== Installation ==


1. Upload the complete `blog-download` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. You can delete the files created at wp-content/plugins/blog-download/downloads/, they are only temporal files created for the conversion, after the user downloads the blog post, they are not required. This can be done automaticly with a cron job.
